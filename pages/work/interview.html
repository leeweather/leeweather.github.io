# 如何获取 url 上 query 的参数，www.baidu.com/?id=123&name=abc&age=18

一定要手写！！！用原生，说 this.$router.query 的让他滚

let query = window.location.href.split('?')[1]

let arr = query.split('&')

arr = arr.map((v) => {
  let keyValue = v.split('=')
  keyValue = keyValue.map((item) => '"' + item + '"')
  return keyValue.join(':')
})

let obj = arr.join(',')
console.log(obj)
console.log(JSON.parse(`{${obj}}`))

# 手写数组去重

// 最最基础的双重遍历
function unique(arr) {
  if (!Array.isArray(arr)) {
    console.log('type error!')
    return
  }
  var array = []
  for (var i = 0; i < arr.length; i++) {
    if (array.indexOf(arr[i]) === -1) {
      array.push(arr[i])
    }
  }
  return array
}
// 高级点的可以用Set 
[...new Set(arr)]

# Vue.$set的作用

因为vue2借助了object.defineProperty监控数据的变化，但是defineProperty无法监控数组长度和对象上key的变化，因此需要$set手动给数据添加响应式

# Vue响应式数据的原理

  • vue2借助Object.defineProperty对数据进行挟持，通过观察者模式，在set的时候通知watcher,watcher收到通知后重新渲染get中的组件，生成新的dom
  • vue3 借助了es6的proxy

# Vuex的属性

  • state 存储数据
  • getter 计算属性
  • mutation 修改state的方法，只能进行同步操作
  • action 存放异步操作
  • module 模块化

# 如何在组件中异步修改state的数据

  1. 在组件中通过vuex的dispatch方法调用action中的方法
  2. 在action中通过vuex的commit方法调用mutation的方法
  3. 在mutation中修改state

# react中函数组件和类组件的区别

类组件拥有自己的生命周期和数据，而函数组件更接近一个纯组件，没有自己的数据，react16.8之后可以借助hooks模拟

# useEffect这个hooks的第二个参数是什么作用，不传会怎么样

如果第二个参数不传，每次渲染都会执行effect中的回调， 如果传【】，则只会在初始化的时候执行一次 如果其他state,则会在state发生改变的时候执行
